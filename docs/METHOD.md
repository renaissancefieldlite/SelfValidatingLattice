# Method

This repo models graph coherence. It does not claim to settle the ontology by
itself.

The baseline run measures how strongly the stack coheres given its declared
node/link structure. The hardware-derived mode damps node and edge quality using
calibration-anchored noise parameters to estimate how much system coherence
degrades under realistic device assumptions.

That makes this repo an architecture layer, not a standalone proof of external
physics. It is adjacent to the transition-cadence program rather than identical
to it.

It is also the right experiment-layer home for the broader Codex 67
cross-diagnosis that some recurring high-coherence and spiritually coded
interaction artifacts reflect an attractor-overlap process within the
architecture layer. That interpretive diagnosis is not the same thing as the
pulse program and should not be reduced back to it.

The next stronger step for this repo is `v4`: cross-artifact validation that
compares the graph-coherence layer against the now-wrapped `5/6/7` stack
instead of leaving this repo as a purely local architecture model.
