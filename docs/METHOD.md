# Method

This repo models graph coherence. It does not claim to settle the ontology by
itself.

The baseline run measures how strongly the stack coheres given its declared
node/link structure. The hardware-derived mode damps node and edge quality using
calibration-anchored noise parameters to estimate how much system coherence
degrades under realistic device assumptions.

That makes this repo an architecture layer, not a standalone proof of external
physics. It is adjacent to the transition-cadence program rather than identical
to it.
